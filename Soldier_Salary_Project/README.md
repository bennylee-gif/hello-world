# 대한민국 군인(병장) 월급 추이 분석 (2010-2025)

## 프로젝트 개요
2010년부터 2025년까지 대한민국 육군 병장의 월급 데이터를 기반으로 임금 상승 추이를 시각화하고, 물가(치킨 가격) 대비 실질 구매력 변화를 분석한 미니 프로젝트입니다.

### 사용 기술 (Tech Stack)
Language: Python 3
Library: Pandas, Matplotlib
Environment: Google Colab

#### 분석 내용
1. 연도별 월급 상승 추이 시각화
Matplotlib을 활용하여 15년간의 급격한 월급 상승 곡선을 꺾은선 그래프로 구현했습니다.
Insight: 2010년 약 9만 원대였던 월급이 2025년 150만 원으로 약 15.3배 상승함을 확인했습니다.
2. '치킨 지수' (구매력) 분석
단순 금액 비교를 넘어, 체감 물가를 반영하기 위해 '치킨 1마리 가격(2만 원 가정)'을 기준으로 파생 변수(Chicken_Count)를 생성했습니다.

#### Insight:
2010년: 월급으로 치킨 약 4.8마리 구매 가능
2025년: 월급으로 치킨 약 75.0마리 구매 가능
결론: 물가 상승을 고려하더라도 병사들의 실질적인 소비 여력이 대폭 개선되었습니다.
